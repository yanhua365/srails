srails
======

基于spring的快速开发框架，像使用rails一样使用spring。

类似于appfuse或springside，但使用更新版本的spring，更简单的架构，更完善的文档。

使用如下技术：

- spring
-spring mvc
-jsp
-sitemesh
-spring data jpa
-spring security
-jquery

目前需实现的功能：
- 提供快速查询功能
  - 基于注解的查询对象
  - data-jpa封装

-配置Spring data jpa 的Domain Converter
-ajax封装
- daja-jpa的审计功能和spring security结合
